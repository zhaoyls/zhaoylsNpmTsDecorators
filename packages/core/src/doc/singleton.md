    # singletonProxy
    返回一个代理，允许类只有一个实例。

    ## Usage

    ``` ts
      import { singleton } from "@zylmo/core"

      const a = singleton.singletonProxy()
      const b = singleton.singletonProxy()

      // do something
    ```
